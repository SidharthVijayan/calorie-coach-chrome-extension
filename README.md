# 🍽️ Calorie Coach – Chrome Extension

Calorie Coach is a Chrome Extension that automatically detects food items on any webpage and estimates their calorie values — helping you track your nutrition effortlessly while browsing.


## 🚀 Features

- 🔍 Detects food items from any webpage
- 🍗 Estimates calorie values instantly
- 📊 Tracks total daily calorie intake
- ⚡ Works across sites (Zomato, blogs, recipes, etc.)
- 💾 Saves data locally using Chrome Storage

---

## 🧠 How It Works

1. The extension scans webpage text
2. Matches food keywords (e.g., pizza, biryani, paneer)
3. Assigns calorie values from a built-in dataset
4. Displays results in a popup UI
5. Stores daily calorie totals locally
